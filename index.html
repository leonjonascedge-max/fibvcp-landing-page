FAQ

Common Questions

Everything you need to know before you buy.

What markets do these systems trade? +
FibVCP v2.6 and WickPrint™ v3.2 are designed for Forex pairs. The Yen Edition is specifically optimised for GBPJPY, EURJPY, AUDJPY, USDJPY, and CADJPY. USDJPY operates as a dual-session pair — Yen Edition rules for Asian/London sessions, standard protocol for New York.
Do I need special software or indicators? +
No proprietary indicators are required. Both systems operate on standard charting platforms (MT4/MT5) using ATR(14), Order Block/FVG identification, and multi-timeframe chart analysis. The methodology is framework-based, not indicator-dependent.
What is the Correlation Protocol v1.0? +
The Correlation Protocol limits you to ≤2 correlated pairs at full size simultaneously. When running 2+ same-direction USD trades, a mandatory USDJPY hedge scan is required. The hedge position is sized at 50% of the main trade's risk. This protocol was developed following a live loss event caused by unhedged correlated exposure.
What changed in WickPrint™ v3.2 vs v3.1? +
v3.2 addressed four confirmed execution leaks from a root-cause audit: (1) mandatory OB/FVG/3×-tested SR validation added, (2) M1/M5 precision entry layer made mandatory, (3) counter-trend Mode 2 hard-prohibited, (4) Mode 1 missed = trade void — no Mode 2 substitution permitted. Additional: 0.5× H1 ATR Mode 2 extension gate, 6-bar staleness cutoff, and 0.3× H1 ATR rectangle minimum.
Is this suitable for beginners? +
The Starter Bundle with the Eclipse EMA Strategy is designed as the on-ramp. The full systems (FibVCP + WickPrint™) require a working knowledge of multi-timeframe analysis, Order Block/FVG concepts, and ATR-based position sizing. Paper trading is strongly recommended before live deployment.
